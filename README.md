# endobio-ai

A modern React + TypeScript web application built with Vite, TailwindCSS, and PostHog analytics.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm

## Installation

1. Clone the repository
2. Install dependencies:

```bash
npm install
```

## Development

Start the development server with hot module reloading:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` by default.

## Build

Build the application for production:

```bash
npm run build
```

This runs TypeScript checks and builds the project with Vite.

## Preview

Preview the production build locally:

```bash
npm run preview
```

## Linting

Check code quality with ESLint:

```bash
npm run lint
```

## Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **TailwindCSS** - Utility-first CSS
- **React Router** - Client-side routing
- **Headless UI** - Unstyled, accessible components
- **Hero Icons** - Icon set

## Analytics

- **Microsoft Clarity** - Session recording and heatmaps
- **Google Analytics** - Traffic and user behavior tracking
- **PostHog** - Product analytics and feature tracking
